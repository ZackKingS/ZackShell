## 1. 设计原则

- **三大区域一屏可达**:会话(左)、工作区(中)、监控(右),对标 FinalShell 但更紧凑。
- **以连接为中心**:选中一个会话,中间和右侧内容随之切换。
- **不喧宾夺主**:监控面板可折叠;终端是主角。
- **中文优先、深色默认**:运维场景常用深色主题。

### 区域占比(默认)
- 会话侧栏:固定 ~240px,可拖拽宽度、可折叠。
- 工作区:自适应填充(主区域)。
- 监控面板:固定 ~300px,可折叠到右边缘。
- 顶栏 ~40px,状态栏 ~28px。

## 3. 各区域详细设计

### 3.1 顶栏(TopBar)
- **左**:应用 Logo + 名称。
- **中**:会话标签页(多连接),每个标签:状态圆点(绿=已连/灰=断开/黄=连接中)+ 会话名 + 关闭按钮;`+` 新建。
- **右**:设置入口 ⚙、窗口控制(最小化/最大化/关闭,自绘标题栏)。

### 3.2 会话侧栏(SessionSidebar)
- 顶部搜索框,按名称/主机过滤。
- 树形分组:文件夹可展开折叠;会话项显示状态圆点 + 名称。
- 交互:
  - 单击选中,双击/回车连接(打开新标签)。
  - 右键菜单:连接、编辑、复制、删除、移动到分组。
  - 底部「+ 新建会话」按钮 → 弹出会话编辑对话框。

### 3.3 工作区(Workspace)
每个会话标签内含子标签:**终端 / 文件**(默认终端)。

**终端视图(TerminalView)**
- 全区域 xterm.js;随窗口自适应(addon-fit)。
- 右键:复制/粘贴/清屏/搜索。
- 顶部细工具条(可选):字号 +/-、搜索、清屏。

**文件管理器(FileManager)**
```
┌─ 路径栏: /home/azureuser            [↑上级] [⟳刷新] [新建▾]─┐
├──────────────┬───────────────────────────────────────────────┤
│ 目录树        │ 名称        大小    类型   修改时间  权限  属主 │
│ ▾ /          │ 📁 .config  -      目录   06-13     drwx  root  │
│   ▸ home     │ 📄 a.log    1.4KB  文件   05-09     -rw-  root  │
│   ▸ etc      │ ...                                            │
└──────────────┴───────────────────────────────────────────────┘
```
- 左目录树 + 右文件列表(Ant Design Table)。
- 列:名称 / 大小 / 类型 / 修改时间 / 权限 / 属主(与 FinalShell 一致)。
- 交互:双击目录进入、双击文本文件在线编辑、拖拽上传、右键(下载/删除/重命名/改权限/属性)。
- 上传/下载进度显示在底部状态栏的传输区。

### 3.4 监控面板(MonitorPanel)
自上而下卡片式,数据由 `monitor:update` 推送,默认 2s 刷新:
1. **系统信息**:主机名、内核、运行时长、负载(1/5/15min)。
2. **CPU**:百分比 + 进度条(多核可展开)。
3. **内存 / 交换**:已用/总量 + 进度条。
4. **网络曲线**:上下行实时折线(uPlot),显示当前 ↑/↓ 速率。
5. **磁盘**:各挂载点 可用/总量 + 进度条。
6. **进程 Top**:小表格,可切换按内存/CPU 排序。
- 整个面板可折叠;无数据时显示骨架屏(首帧差值未就绪)。

### 3.5 状态栏(StatusBar)
连接状态、延迟、当前传输任务进度、编码、终端尺寸。

---

## 4. 对话框 / 弹层

### 4.1 会话编辑对话框
- 字段:名称、分组、主机、端口(默认 22)、用户名、认证方式(密码 / 密钥)。
  - 密码:输入框(存入凭据库,不回显明文)。
  - 密钥:选择私钥文件 + passphrase。
- 高级:跳板机(选已有会话)、keepalive、编码、终端类型。
- 按钮:测试连接、保存、保存并连接。

### 4.2 主机密钥确认
- 首次连接弹出指纹确认;指纹变更时红色告警。

### 4.3 设置
- 外观:主题(深/浅)、字体、字号、语言(中/英)。
- 终端:滚动行数、光标样式。
- 监控:采集频率。
- 安全:主密码设置。

---

## 5. 主题与视觉

- 默认深色,配色参考 VS Code Dark+ / FinalShell。
- 终端配色方案可选(One Dark、Solarized 等)。
- 强调色用于状态圆点与进度条(绿=正常、黄=警告、红=危险,如磁盘>90%标红)。
- 字体:UI 用系统默认;终端用等宽(Cascadia Code / Consolas)。

---

## 6. 响应式与可用性

- 窗口缩小到一定宽度时,监控面板自动折叠;再小则会话侧栏可折叠为图标条。
- 全键盘可达:Ctrl+T 新建、Ctrl+W 关闭标签、Ctrl+Tab 切换、Ctrl+F 搜索。
- 所有耗时操作(连接、传输)有 loading / 进度反馈。
